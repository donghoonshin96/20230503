CSS적용은 element.style.property형식으로 접근 및 변경이 가능

  • property는 자바스크립트의 코드 : -(하이픈) 적용안됨, 띄어쓰기 적용안됨
  • css의 속성의 -(하이픈)를 제거하고, 카멜표기법9뒤에 오는 글자를 대문자로 변경) 으로 변경
  • Element.setAttribute(name, value) : Element에 속성 부여 이미 있는 속성이면 업데이트, 없는 속성이면 새속성으로 추가
  • CssStyleDeclaration.setProperty(propertyName, value) : css 스타일을 부여할 대상에게 css프로퍼티 부여함 propertyName : Css 속성 이름, value : 속성 값

색상명을 입력하고, 색상변경 버튼을 클릭하면 현재 h1 태그의 배경색이 입력한 색상이 되게 변경하기 style.backgroundcolor X
